## Features
- Html and form helpers
- Twitter Bootstrap
- bower, angularjs with requirejs
- Heroku, mongodb, travis, new relic, nodetime
- karma with mocha + chai + requirejs for tests
- protractor for angularJS e2e tests
- Facebook login with oauth.io
